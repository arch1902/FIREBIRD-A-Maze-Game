Pacman written in C++.

## Installation

> cd pacman-sdl
> make
```

## Usage

```console
> ./pacman-sdl --help
Usage: pacman-sdl [options]

Options:
    -h, --help          print this help menu
> ./pacman-sdl  # normal mode
> ./pacman-sdl -f  # fullscreen mode
```

### Key Config

* Player1
  * UP, LEFT, DOWN, RIGHT: ↑ ← ↓ →
* Player2
  * UP, LEFT, DOWN, RIGHT: W A S D
